# gulpApp-example

[![Build Status](https://travis-ci.org/SuperOleg39/gulpApp-example.svg?branch=master)](https://travis-ci.org/SuperOleg39/gulpApp-example)

Install project:
```
git clone https://github.com/SuperOleg39/gulpApp-example.git
```
```
npm install
```
```
gulp build
```

Watch changes:
```
gulp webserver
```

Create a server:
```
gulp webserver
```

Clean build project:
```
gulp clean
```


###Or all together
```
gulp
```
